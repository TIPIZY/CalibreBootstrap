# CalibreBootstrap
Bootstrap template for the Calibre Content Server

![Imgur](http://i.imgur.com/AQGbNEh.png)

# Install
- git clone into ~/.config/calibre/  or copy contents into ~/.config/calibre/
- Restart calibre-server

# Libraries in use
- jQuery 1.12.4
- Bootstrap 3.3.7
- readmore.js
- FontAwesome
